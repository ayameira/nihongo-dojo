# Nihongo Dojo

Nihongo Dojo is a personalized language learning application that combines an AI-powered conversational tutor with vocabulary and grammar tracking. Japanese is the flagship language, with English and French also supported; each is a separate room with its own vocabulary, grammar, sessions, and memory of the student. The tutor runs on any configured provider — Groq, Gemini, OpenRouter, or another OpenAI-compatible endpoint — streaming responses in real time while maintaining persistent learning context across sessions. It adapts to the student's skill level based on explicit difficulty feedback.

The application features a React/TypeScript frontend with a three-pane layout: a collapsible left sidebar for vocabulary browsing and multi-session chat management, a central chat interface with markdown rendering and image attachment support, and a right sidebar displaying budget tracking with an interactive Tanuki samurai mascot. The FastAPI backend provides async API endpoints for chat streaming via Server-Sent Events, vocabulary and grammar CRUD, student facts, text-to-speech, and telemetry. SQLite stores vocabulary entries (with status tracking for New/Learning/Mature words), grammar points with level and assessment state, chat sessions and messages, student facts, and token usage logs for cost monitoring.

Long-term memory lives in the `student_facts` table rather than in files. Facts about the student's goals, background, and preferences are written both by the tutor's `manage_student_facts` tool and by a background listener agent that reads conversations as they happen, while a separate background assessor updates grammar mastery. Combined with conversation memory compaction for long sessions and configurable weekly spending limits, this provides a sustainable, context-aware tutoring experience that grows with the learner.

A key differentiator is the deep Anki integration: a setup wizard reads your existing collection, lets you select decks and map fields, and imports vocabulary with intelligent status mapping based on Anki's spaced repetition intervals — so chat practice uses the words you are actually studying. Your Anki collection is only ever read, never modified.
