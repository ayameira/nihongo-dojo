# Nihongo Dojo

Nihongo Dojo is a personalized Japanese language learning application that combines an AI-powered conversational tutor with vocabulary tracking and study notes management. Built for intermediate learners, the application provides real-time Japanese practice through streaming chat powered by Google Gemini, while maintaining persistent learning context across sessions. The AI tutor adapts to the student's skill level based on explicit difficulty feedback and leverages tools to autonomously update class notes and student records, creating a continuously improving learning experience.

The application features a React/TypeScript frontend with a three-pane layout: a collapsible left sidebar for vocabulary browsing and multi-session chat management, a central chat interface with markdown rendering and image attachment support, and a right sidebar displaying budget tracking with an interactive Tanuki samurai mascot. The FastAPI backend provides async API endpoints for chat streaming via Server-Sent Events, vocabulary CRUD operations, notes management, and telemetry tracking. SQLite stores vocabulary entries (with status tracking for New/Learning/Mature words), chat sessions and messages, and token usage logs for cost monitoring.

A key differentiator is the deep Anki integration that automatically syncs with the user's existing flashcard collection on startup, importing vocabulary with intelligent status mapping based on Anki's spaced repetition intervals. The system maintains two levels of memory: short-term "class notes" tracking current study focus and recent corrections, and long-term "student records" capturing goals, background, and learning preferences. Combined with conversation memory compaction for long sessions and configurable weekly spending limits, Nihongo Dojo provides a sustainable, context-aware tutoring experience that grows with the learner.
